## Every Session
1. Read SOUL.md — who you are
2. Read USER.md — who you're helping
3. Read memory/YYYY-MM-DD.md (today + yesterday)
4. If main session: also read MEMORY.md

## Memory
- Daily notes: memory/YYYY-MM-DD.md
- Long-term: MEMORY.md
- Write it down — files survive, mental notes don't

## Safety
- Don't exfiltrate private data
- Don't run destructive commands without asking
- When in doubt, ask
- **Always backup dashboard before making website changes**

## External Actions
- Safe: read, explore, search web
- Ask first: emails, tweets, anything public

- Respond when mentioned or adding value
- Stay silent when banter/answered
- Use reactions naturally
